## 前書き
最近、Xで嫌な動きがありました。しかし、このようなことは他のプラットフォームでも起こりうるため、このツールを作成しました。  
正直、**少し手間がかかり**、**とても簡素なツール**です。もし「こんなもの作ってどうするの？」と言われても笑って受け入れます。  

Cloudflare Pagesは現在無料ですが、将来的な保証はありません。また、セキュリティについても自己責任で確認してください。私はセキュリティの専門家ではありません。  
この.exeファイルは、選択したフォルダ（フォルダのみ選択可能）の画像を、単一ページの静的画像展示サイト用に変換して出力します。Cloudflare Pagesへのデプロイはご自身で行ってください。  

GitHubとCloudflare Pagesの2つのアカウントが必要です。GitHub未使用の方は少し大変かもしれません。GitHub Desktopを使えば、ローカルから簡単にアップロード・編集できます。  

この手間をかけることで得られる効果は…一つのURLで、SNSに投稿するとサムネイルが表示され、クリックすると大きな画像と声明文が見られます（カスタマイズ可能）。右クリックによるダウンロードは制限されますが、スクリーンショットや印刷など、完全に防ぐことはできません。それでも、Xが画像を取得する場合でも、低解像度のサムネイルしか取得できません。  

---

Image Defense Toolは、**画像展示および保護用のツール**で、作者が画像を公開する際に、無断利用やAI訓練への使用を防ぎつつ、静的な展示ページを素早く生成できます。  

GUI（EXE）操作で、プログラム知識がなくても使用可能です。

---

## 目的と機能

### 主要目的
- 画像がAIモデルの訓練に直接使用されるのを防ぐ
- クリエイターのオリジナル作品を保護
- 画像の安全な表示を提供
- 迅速に展開可能な画像展示ページを作成

### 主要機能
- 展示用画像を自動生成
- 自動で裁切可能なサムネイルを生成
- 単一ページの静的画像展示サイトを生成
- GitHub Pages / Cloudflare Pagesへの直接デプロイ対応

## 操作手順（GUI版）

### 1️⃣ プログラム起動
実行：
Image Defense Tool.exe


---

### 2️⃣ 画像フォルダ選択
- **Browse** をクリック
- 画像が入ったフォルダを選択（JPG / PNG）

---

### 3️⃣ サムネイル設定（Thumbnail）
- **アスペクト比**
  - `original`：元の比率を維持
  - `1:1` / `16:9` / `4:3`：自動裁切
- **最大サイズ**
  - サムネイルの最大辺の長さ（ピクセル）

---

### 4️⃣ サイト設定
- **Site Title**：展示ページのタイトル
- **Base URL**：デプロイ先URL（Open Graph / 共有用）

---

### 5️⃣ 声明設定（Notice）
- 画像使用声明を表示するか選択可能
- 以下をカスタマイズ可能：
  - Title
  - Body
  - Extra

---

### 6️⃣ 展示ページ作成
- **Build Website** をクリック
- 静的ウェブサイト用のフォルダが出力されます

---

## 出力構造
最終出力フォルダ構造：
MG/  
├─ index.html # メイン展示ページ  
├─ image.bat # 画像ローカル閲覧用  
├─ og_preview.jpg # SNS共有プレビュー  
├─ viewer.js # 画像表示・操作スクリプト  
└─ style.css # スタイルシート  


📌 **補足**
- `index.html`：メインページ
- `og_preview.jpg`：SNS共有時のプレビュー画像
- `viewer.js`：画像表示・操作用
- `style.css`：デザイン・表示スタイル

---

## デプロイ方法簡略

### GitHub Pages
1. GitHubリポジトリを作成
2. `IMG/` 内のファイルをアップロード
3. Pagesを有効化（Rootまたは `/docs`）

### Cloudflare Pages
1. Pagesプロジェクト作成
2. `IMG/` を出力ディレクトリとしてアップロード
3. 追加ビルド操作は不要
